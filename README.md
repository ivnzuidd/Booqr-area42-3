# Booqr — Area52

> Reserveringssysteem voor vakantiepark Area52 | Semester 2 groepsproject

---

## Projectbeschrijving

Booqr is een webapplicatie waarmee gasten van vakantiepark Area52 faciliteiten kunnen reserveren. Dit betreft de restaurant en snackbar in het vakantiepark. Het systeem biedt klanten de mogelijkheid om in te loggen, een reservering te maken en deze te beheren.

Dit project wordt ontwikkeld als onderdeel van het groepsproject voor smester 2, de propedeusefase.

## Database model

De applicatie gebruikt de volgende entiteiten:

- **Klant** — parkgasten die reserveringen maken
- **LogIn** — inloggegevens gekoppeld aan een klant (1:1)
- **Reservering** — een boeking van een faciliteit door een klant (1:N)
- **Faciliteit** — te boeken voorziening op het park, bijv. restaurant (N:1)
